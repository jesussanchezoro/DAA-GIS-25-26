# Diseño y Análisis de Algoritmos (DAA - GIS)

Bienvenido al repositorio central de la asignatura **Diseño y Análisis de Algoritmos** del **Grado en Ingeniería del Software**. Este espacio contiene las implementaciones, ejemplos y ejercicios prácticos desarrollados para el curso.

El objetivo de este repositorio es proporcionar una base sólida para entender la eficiencia algorítmica, desde el análisis teórico hasta la implementación práctica.

---

## Estructura del Repositorio

El material está organizado por unidades temáticas para facilitar su localización:

* **`0_practicas/`**: Ejercicios de prácticas resueltos en clase.
* **`2_voraces/`**: Código hecho en clase del tema de algoritmos voraces.
* **`3_grafos/`**: Código hecho en clase del tema de algoritmos sobre grafos.
* **`4_voraces_grafos/`**: Código hecho en clase del tema de algoritmos voraces sobre grafos.
* **`5_dyv/`**: Código hecho en clase del tema de divide y vencerás.
* **`6_vuelta_atras/`**: Código hecho en clase del tema de vuelta atrás.
